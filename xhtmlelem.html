<big></big> Großschrift, zeichnet einen Text größer als normal aus Inline semantisch
<blockquote></blockquote> Zitat in Form eines eigenen Textabsatzes Block semantisch
<body></body> Inhaltsbereich eines HTML-Dokuments
<br /> Erzwungener Zeilenumbruch an der betreffenden Stelle Inline
<button></button> Schaltfläche in einem Formular Inline
<caption></caption> Tabellenüberschrift / -unterschrift semantisch
<cite></cite> Zitat innerhalb eines Absatzes Inline semantisch
<code></code> Quelltext-Auszug innerhalb eines Absatzes Inline semantisch
<col /> Tabellenspalte
<colgroup></colgroup> Gruppe von Tabellenspalten
<dd></dd> Definition, Beschreibung, detaillierter Inhalt zu einem dt-Element semantisch
<del></del> gelöschter Inhalt (Änderungsmarkierung) Inline und Block
<dfn></dfn> Definition innerhalb eines Absatzes Inline semantisch
<div></div> Allgemeiner Bereich Block
<dl></dl> Liste mit dt- und dd-Elementen, für Definitionslisten, Dialoge usw. Block semantisch
<dt></dt> Zu definierender Ausdruck in einer Definitionsliste, Sprechername in einem Dialog usw. semantisch
<em></em> emphatischer Ausdruck innerhalb eines Absatzes (z.B. überspitzte Formulierung) Inline semantisch
<fieldset></fieldset> Gruppe von Elementen in einem Formular, Reiter in einem Dialog Block
<form></form> Formular, Dialog Block

<img /> Referenzierte und eingebettete Grafik Inline
<input /> Formularelement, verwendbar für diverse Typen wie Eingabefelder, Schaltflächen, Auswahlfelder usw. Inline
<ins></ins> Eingefügter Inhalt (Änderungsmarkierung) Inline und Block
<kbd></kbd> Darstellung eines über Tastatur einzugebenden Textes im Text Inline semantisch
<label></label> Beschriftungstext eines Formularfeldes Inline
<legend></legend> Überschrift für eine fieldset-Gruppe von Elementen in einem Formular
<li></li> Listenpunkt in einer ungeordneten oder einer geordneten Liste semantisch
<link /> Logische Beziehungen der aktuellen Webseite zu anderen Webseiten
<map></map> Bereich für verweis-sensitive Grafiken Inline
<meta /> Meta-Angabe für nicht im Inhalt sichtbare Dokumentinformationen oder für Anweisungen an den Webserver
<noscript></noscript> Bereich für Inhalte, die angezeigt werden, wenn JavaScript im Browser deaktiviert oder nicht verfügbar ist Block
<object></object> Eingebettetes Objekt, geeignet für spezielle Multimedia-Objekte, Java-Applets usw. Inline
<ol></ol> Geordnete (z.B. nummerierte) Liste Block semantisch
<optgroup></optgroup> Gruppe von Auswahlmöglichkeiten in einer Auswahlliste
<option></option> Auswahlmöglichkeit in einer Auswahlliste
<p></p> Textabsatz Block semantisch
<param /> Parameter eines Objekts bei object-Element
<pre></pre> Präformatierter Text (alle Whitespace-Zeichen werden im Browser so dargestellt wie im Text enthalten) Block
<q></q> Wörtliche Rede in einem Text Inline semantisch
<samp></samp> Beispieltext in einem dokumentierenden Text Inline semantisch
<script></script> Scriptbereich für JavaScript und andere Scriptsprachen Inline
<select></select> Auswahlliste in einem Formular Inline
<small></small> kleiner dargestellter Text als Normalschrift Inline semantisch
<span></span> allgemeiner Bereich innerhalb eines Absatzes Inline
<strong></strong> stark betonter Inhalt im Text Inline semantisch
<style></style> Bereich für Stylesheet-Definitionen, z.B. in CSS
<sub></sub> Tiefgestellter Text Inline semantisch
<sup></sup> Hochgestellter Text Inline semantisch

<td></td> Tabellenzelle semantisch

<thead></thead> Kopfbereich einer Tabelle semantisch
<title></title> Titel eines HTML-Dokuments
<tr></tr> Tabellenzeile semantisch
<ul></ul> Unsortierte Liste (Aufzählungsliste) Block semantisch
<var></var> Variabler Name in einem dokumentierenden Text Inline semantisch
